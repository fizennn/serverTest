============================
DANH SÁCH API MODULE POST
============================

1. Tạo bài viết mới (Admin)
- Endpoint: POST /posts
- Body:
  {
    "title": "Tiêu đề bài viết",
    "content": "Nội dung bài viết",
    "images": ["url1", "url2"],
    "attachedProducts": ["productId1", "productId2"],
    "tags": ["tag1", "tag2"]
  }
- Response: Thông tin bài viết vừa tạo
- Quyền truy cập: Chỉ admin

2. Lấy danh sách bài viết
- Endpoint: GET /posts
- Query: page, limit, search, tag, featured, sortBy, sortOrder
- Response: Danh sách bài viết, phân trang
- Quyền truy cập: Ai cũng dùng được

3. Lấy danh sách bài viết nổi bật
- Endpoint: GET /posts/featured
- Response: Danh sách bài viết nổi bật
- Quyền truy cập: Ai cũng dùng được

4. Lấy chi tiết bài viết
- Endpoint: GET /posts/:id
- Params: id
- Response: Thông tin chi tiết bài viết
- Quyền truy cập: Ai cũng dùng được

5. Like/Unlike bài viết
- Endpoint: POST /posts/:id/like
- Params: id
- Response: { message, likeCount, isLiked }
- Quyền truy cập: Người dùng đã đăng nhập

6. Lấy danh sách người like bài viết
- Endpoint: GET /posts/:id/likes
- Params: id
- Response: { likeCount, likes }
- Quyền truy cập: Ai cũng dùng được

7. Thêm comment vào bài viết
- Endpoint: POST /posts/:id/comments
- Params: id
- Body: { content, images }
- Response: Thông tin comment vừa tạo
- Quyền truy cập: Người dùng đã đăng nhập

8. Lấy danh sách comment của bài viết
- Endpoint: GET /posts/:id/comments
- Params: id
- Query: page, limit
- Response: Danh sách comment, phân trang
- Quyền truy cập: Ai cũng dùng được

9. Cập nhật comment
- Endpoint: PUT /posts/:postId/comments/:commentId
- Params: postId, commentId
- Body: { content, images }
- Response: Thông tin comment sau cập nhật
- Quyền truy cập: Người dùng đã đăng nhập (chỉ sửa comment của mình), admin sửa được tất cả

10. Xóa comment
- Endpoint: DELETE /posts/:postId/comments/:commentId
- Params: postId, commentId
- Response: { message }
- Quyền truy cập: Người dùng đã đăng nhập (chỉ xóa comment của mình), admin xóa được tất cả

11. Cập nhật bài viết (Admin)
- Endpoint: PUT /posts/:id
- Params: id
- Body: như tạo bài viết
- Response: Thông tin bài viết sau cập nhật
- Quyền truy cập: Chỉ admin

12. Xóa bài viết (Admin)
- Endpoint: DELETE /posts/:id
- Params: id
- Response: { message }
- Quyền truy cập: Chỉ admin

13. Lấy tất cả bài viết (Admin)
- Endpoint: GET /posts/admin/all
- Query: page, limit, status
- Response: Danh sách tất cả bài viết
- Quyền truy cập: Chỉ admin

14. Bật/tắt trạng thái nổi bật (Admin)
- Endpoint: PUT /posts/:id/toggle-featured
- Params: id
- Response: { message, isFeatured }
- Quyền truy cập: Chỉ admin

15. Bật/tắt trạng thái hiển thị (Admin)
- Endpoint: PUT /posts/:id/toggle-status
- Params: id
- Response: { message, isActive }
- Quyền truy cập: Chỉ admin

16. Thống kê bài viết (Admin)
- Endpoint: GET /posts/admin/statistics
- Response: Thống kê tổng quan về bài viết
- Quyền truy cập: Chỉ admin
